# MyBlog

MyBlog is a modern full-stack blogging platform that enables users to create, edit, and manage posts with a clean UI and secure backend powered by Appwrite.

## 🚀 Features

- 🔐 Authentication (Login / Signup)
- ✍️ Rich Text Editor for writing posts
- 🖼️ Image upload & preview
- 📝 Create, Edit, Delete posts
- 📄 Slug-based routing for SEO-friendly URLs
- 🧭 Protected routes for authenticated users
- 🎨 Responsive dark-themed UI with Tailwind CSS

## 🛠️ Tech Stack

- **Frontend:** React (Vite), Tailwind CSS
- **Backend:** Appwrite (Auth, Database, Storage)
- **Routing:** React Router
- **Forms:** React Hook Form
- **Editor:** TinyMCE
